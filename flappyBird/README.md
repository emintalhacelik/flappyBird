# Flappy Bird Game (C# Windows Forms)

This is a simple Flappy Bird game that I made using C# and Windows Forms.

## About the Game

In this game, you control a bird and try to pass between the pipes.  
If the bird hits a pipe or the ground, the game is over.  
Each time you pass a pipe, your score increases.

## How to Play

- Press **Space** to make the bird go up.
- When you release the key, the bird goes down because of gravity.
- Try to survive as long as possible and get a high score.

## Technologies Used

- C#
- Windows Forms
- Visual Studio
