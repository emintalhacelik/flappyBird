# flappyBird

This project is a simple Flappy Bird clone developed using C#.
The game recreates the basic mechanics of the classic Flappy Bird game where the player controls a bird and tries to pass through pipes without hitting them.

## 🎮 Gameplay

* Press **Space** to make the bird jump.
* Avoid hitting the pipes.
* Each pipe you pass increases your score.
* If the bird hits a pipe or the ground, the game ends.

## 🛠 Technologies Used

* C#
* Windows Forms
* Visual Studio

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project in Visual Studio.
3. Run the project.
